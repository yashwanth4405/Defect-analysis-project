# 📊 Defect Analysis Project (Manufacturing RCA)

## 🔍 Overview
This project focuses on analyzing manufacturing defects and identifying root causes using data analytics techniques. The goal is to reduce defect rates and improve production efficiency.

---

## 🎯 Objectives
- Analyze defect patterns across plants and machines
- Identify key root causes of defects
- Perform data-driven decision making
- Build interactive dashboards for monitoring

---

## 🛠️ Tools & Technologies Used
- **Excel** – Data cleaning & preprocessing
- **SQL** – Data extraction & querying
- **Python** – Regression & clustering analysis
- **Power BI** – Interactive dashboard & visualization

---

## 📂 Project Structure
/excel → Q1, Q2 analysis
/sql → Q3 & Q4 queries
/python → Regression & Clustering models
/powerbi → Dashboard file (.pbix)
/screenshots → Output visuals
/docs → Supporting documents


---

## 📊 Key Analysis Performed

### 1. Excel Analysis
- Data cleaning and transformation
- KPI calculations
- Trend analysis

### 2. SQL Analysis
- Defect frequency queries
- Machine-wise analysis
- Root cause extraction

### 3. Python Analysis
- **Regression Model** → Predict defect rate
- **Clustering** → Identify defect patterns

### 4. Power BI Dashboard
- Defect trends over time
- Plant-wise comparison
- Machine performance insights
- Root cause visualization

---

## 📈 Key Insights
- Certain machines contribute to higher defect rates
- Specific plants show recurring defect patterns
- Seasonal trends impact production quality
- Data-driven monitoring improves decision-making


---

## 🚀 How to Use
1. Open Excel files for initial analysis  
2. Run SQL scripts in database  
3. Execute Python notebooks  
4. Open Power BI file for dashboard  

---

## 📌 Conclusion
This project demonstrates how data analytics can be used to identify root causes in manufacturing and improve operational efficiency.
